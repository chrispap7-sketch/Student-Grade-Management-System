📚 Student Grade Management System

📋 Πρόγραμμα Διαχείρισης Βαθμολογίας Μαθητών

🌟 Περιγραφή

Ένα ολοκληρωμένο σύστημα διαχείρισης βαθμών μαθητών γραμμένο σε C++. Το πρόγραμμα επιτρέπει την καταχώρηση νέων μαθητών, την εισαγωγή και ενημέρωση βαθμών σε πολλά μαθήματα, τον υπολογισμό μέσου όρου και την αποθήκευση δεδομένων σε αρχείο CSV.

✨ Λειτουργίες

✅ Προσθήκη νέων μαθητών με ID, όνομα και επώνυμο
✅ Εισαγωγή βαθμών σε πολλαπλά μαθήματα
✅ Αποθήκευση δεδομένων σε μορφή CSV για ευκολία
✅ Υπολογισμός μέσου όρου βαθμολογίας
✅ Πολλαπλή εισαγωγή βαθμών στην ίδια συνεδρία
✅ Εμφάνιση στοιχείων μαθητή με όλους τους βαθμούς
🛠️ Τεχνολογίες

Γλώσσα: C++
Αποθήκευση δεδομένων: CSV αρχείο
Συλλογή δεδομένων: Δομές (structs) και διανύσματα
Επεξεργασία αρχείων: fstream, sstream
📁 Δομή Αρχείου

text

🤝 Συνεισφορά

Ο κώδικας είναι ανοιχτός για βελτιώσεις. Μπορείτε να:

Προσθέσετε επιπλέον μαθήματα
Βελτιώσετε τον υπολογισμό του μέσου όρου
Προσθέσετε δυνατότητα διαγραφής μαθητών
Υλοποιήσετε γραφική διεπαφή χρήστη
📝 Σημειώσεις

Οι βαθμοί πρέπει να είναι μεταξύ 0 και 10
Το πρόγραμμα διατηρεί την υπάρχουσα βαθμολογία κατά την ενημέρωση
Μπορείτε να εισάγετε βαθμούς σε πολλαπλά μαθήματα χωρίς επανεκκίνηση
